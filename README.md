# Maze Game

A Python maze game using Pygame and Pygame Menu.

## Description

This project is a maze game where you can play yourself or watch AI solve the maze. It includes features like:
- Random maze generation.
- Player movement.
- Auto-play using A* and BFS algorithms.
- Leaderboard.
- Save/Load game.
- Customizable settings (Music, Sound effects).

## Requirements

- Python 3.x
- pygame
- pygame-menu

## Installation

1.  Clone the repository.
2.  Install the required packages:

    ```bash
    pip install pygame pygame-menu
    ```

## Usage

To run the game, execute the `run.py` script from the root directory:

```bash
python run.py
```

## Controls

-   **Arrow Keys**: Move the player.
-   **H**: Get a hint (shows path to a decision point).
-   **R**: Toggle between A* and BFS in auto-play mode.

## Structure

-   `src/`: Contains the source code.
-   `data/`: Contains assets (images, sounds, fonts, database).
-   `run.py`: Entry point for the application.
